# Portal Educativo

A continuación se presenta el trabajo para el módulo 3 del curso de Front End. Es una pequeña aplicación interactiva que funciona como una herramienta para profesores. Toda la interacción se hace a través de la consola del navegador usando ventanas de mensajes y alertas. Comienza con un menú de 4 opciones: 1) calculadora de promedios con las 4 operaciones de aritmética; 2) listado de alumnos; 3) filtro de alumnos; 4) atrás.

## Lo que incluye el proyecto:

- Uso de variables y condicionales para validar que los datos que se ingresan sean correctos.
- Un menú interactivo hecho con un bucle `while` para poder usar la herramienta varias veces sin que se cierre.
- Ciclos `for` para buscar y filtrar información (como las notas de los alumnos).
- Funciones separadas para organizar mejor el código (por ejemplo, sumar, restar, multiplicar y dividir).
- Un arreglo con información de varios alumnos y sus calificaciones para poder hacer las pruebas.
